# Backoffice Admin Panel

## Project info

A Vite + React + TypeScript project with a storefront and an admin backoffice.

## Getting started

```sh
npm install
npm run dev
```

- User dashboard: `http://localhost:5173/`
- Admin login: `http://localhost:5173/admin-login`

## Build

```sh
npm run build
npm run preview
```

## Tech stack

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
